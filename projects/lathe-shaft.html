Lathe Turned Shaft

Machining Project | Centre Lathe | Precision Measurement

Overview

This project involved manufacturing a stepped shaft using a centre lathe. The work included facing, turning, and dimensional inspection.

Project Specifications

Machine Centre Lathe
Material Mild Steel
Chuck Type 3 Jaw Chuck
Operations Facing, Turning, Measurement

Manufacturing Process

  1. Facing the end of the raw stock to create a flat datum surface.
  2. Rough turning the shaft diameter to remove excess material.
  3. Step turning to achieve the required diameters.
  4. Measuring with vernier calipers to ensure dimensional accuracy.
  5. Final finishing pass to improve surface finish.
